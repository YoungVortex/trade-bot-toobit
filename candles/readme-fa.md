تولیدکننده نمودار حرفه‌ای با قابلیت‌های **Smart Money Concepts**، **Triple SuperTrend**، **RSI** و **Pivot Points** — خروجی PNG با کیفیت بالا.

## قابلیت‌ها

### Triple SuperTrend
- **ST 1**: طول 10، ضریب 1.0 (خط توپر)
- **ST 2**: طول 10، ضریب 2.0 (خط خط‌چین)
- **ST 3**: طول 10، ضریب 3.0 (خط نقطه‌چین)
- سبز = روند صعودی، قرمز = روند نزولی

### RSI (شاخص قدرت نسبی)
- RSI با دوره 14 در پنل جداگانه
- خطوط مرجع اشباع خرید (70) / اشباع فروش (30)
- وضعیت لحظه‌ای: اشباع خرید / اشباع فروش / خنثی

### نقاط پیوت
- پیوت بالا (علائم نارنجی ▼ + خط خط‌چین)
- پیوت پایین (علائم سبز ▲ + خط خط‌چین)
- 5 کندل برگشت در هر طرف

### مفاهیم Smart Money
- **BOS** (شکست ساختار) — برچسب‌های آبی/قرمز
- **CHoCH** (تغییر ماهیت) — برچسب‌های فیروزه‌ای/نارنجی
- **بلوک‌های سفارش** — ناحیه صعودی (آبی) / ناحیه نزولی (قرمز)
- **شکاف‌های ارزش منصفانه** — ناحیه صعودی (فیروزه‌ای) / ناحیه نزولی (قرمز)
- خطوط ساختار متصل‌کننده نقاط سوئینگ به شکست‌ها

## سفارشی‌سازی

فایل `candles.py` را ویرایش کنید تا پارامترهای اندیکاتورها را تغییر دهید:

```python
# تنظیمات SuperTrend
st1, d1 = supertrend(df, 10, 1.0)  # طول، ضریب
st2, d2 = supertrend(df, 10, 2.0)
st3, d3 = supertrend(df, 10, 3.0)

# دوره RSI
rv = rsi(df["Close"], 14)

# برگشت پیوت
ph, pl = pivots(df, 5, 5)  # کندل‌های چپ و راست

# طول سوئینگ Smart Money
sm = smart_money(df, 5)
```
